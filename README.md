# generator-rnapp

## Installation
First, you need to install [Yeoman](http://yeoman.io) and generator-rnapp with [npm](https://www.npmjs.com/)

```bash
npm install -g yo
npm install -g generator-rnapp
```

Then generate your new React Native project:

```bash
yo rnapp
```
